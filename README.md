# Misuo Sales Dashboard

An interactive Power BI dashboard designed to track, analyze, and visualize key sales performance metrics, regional distributions, and product trends for Misuo. 

---

## 📊 Dashboard Overview
<img width="1308" height="497" alt="Screenshot 2026-06-08 113124" src="https://github.com/user-attachments/assets/19d37275-6aaf-4cc4-bb08-ecd917917e79" />

Based on the interface layout in **Screenshot 2026-06-08 113124_2.png**, the dashboard delivers a comprehensive view of business health by tracking total revenue, order execution efficiency, and consumer preferences. It features an intuitive, dark-themed sidebar panel for dynamic filtering by client name and delivery/shipping status.

### Key Performance Indicators (KPIs)
* **Total Sales (إجمالي المبيعات):** 1,165,307
* **Total Orders (عدد الطلبات):** 112
* **Average Selling Price (متوسط سعر البيع):** 10,405
* **On-Time Delivery Rate (نسبة الطلبات المسلمة في الموع):** 59%
* **Cancellation Rate (نسبة الطلبات الملغاه):** 41%

### Core Analytics & Visualizations
* **Top Selling Cities (المدينة الأكثر مبيعاً):** Regional sales volume distribution tracking performance across major cities like Cairo and Giza.
* **Product Preferences (المنتج الأكثر مبيعاً & اللون الأكثر مبيعاً):** Granular insights into the most popular inventory types (e.g., beds, sofas) and color variants (e.g., black, white).
* **Platform Performance (عدد المنتجات المباعة حسب المنصة):** A breakdown of order volume processed through various sales channels and social platforms.
* **Customer Segment Breakdown (عدد الأفراد والشركات):** A comparative view showing business distribution between individual consumers (87%) and corporate clients (13%).

## 🛠️ Built With

* **Power BI Desktop** - Data modeling, DAX calculations, and report visualization.
* **Power Query** - Data extraction, transformation, and loading (ETL).
